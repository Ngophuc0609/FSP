# SKILL: DESIGN UI TOKENS (`Skill-Design-UI-Tokens`)

## 1. Purpose & Trigger
Use this skill when tasked with creating design system tokens, wireframes, or accessibility specifications.

## 2. Execution Steps
1. **Define Token Hierarchy:** Specify colors, typography scale (`Inter` / `Roboto`), spacing, and border radii.
2. **Validate Field Usability:** Ensure buttons meet `48x48 logical pixels` and colors remain high-contrast in direct sunlight.
3. **Document Wireframes:** Generate descriptive layout sections using `docs/governance/markdown_convention.md`.
