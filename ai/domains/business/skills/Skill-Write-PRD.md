# SKILL: WRITE REQUIREMENTS SPECIFICATION (`Skill-Write-PRD`)

## 1. Purpose & Trigger
Use this skill when tasked with authoring Product Requirements Documents (`docs/business/prd.md`), BRDs, or FSDs.

## 2. Execution Steps
1. **Review Ontology:** Consult `ai/ontology/business.md` for exact terminology.
2. **Structure Document:** Include Executive Summary, Problem Statement, Target Personas (`Marcus`, `Sarah`), Functional Requirements Matrix, and SLAs.
3. **Map State Machines:** Include Mermaid diagrams illustrating object lifecycle transitions (`Draft` -> `Assigned` -> `InProgress` -> `Completed`).
